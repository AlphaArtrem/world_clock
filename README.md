# World Clock

## Description
A simple world clock made to learn flutter basics.

## Installing 

1. Clone the repository

```
git clone https://github.com/AlphaArtrem/world_clock.git
```

2. Get dependencies

```
pub get
```

## Packages Used

```
http                : ^0.12.0+4
intl                : ^0.16.1
flutter_spinkit     : "^4.1.2"
```
